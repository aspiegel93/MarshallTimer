About the Marshall Timer

The Marshall Timer is a countdown timer created to time classroom exercises, quizzes, and exams. It can also be used to time group meetings, presentations, breaks, and numerous other business school activities.

You can set the timer and have it run on your screen or to a projector. This timer will enable accurate timing, and also provides preset warnings.

A sound is associated with each alert, as well as with the final "Times Up!" message. Click the "preview" link to hear the alert sounds.

Using the Marshall Timer

If the set-up area is not open, click Set Timer to open it. Use these steps to set a countdown time:

  1. For Set Timer, enter the total time of the exam or event in hours and minutes.
  2. Optional. To set Alert One, enter a time, and check Play Sound? to add an audio reminder.
  3. Optional. Set Alert Two and Alert Three.
  4. When finished, click the button Set Timer & Close Settings.
  5. When you're ready to begin, click Start.

Use the following buttons to manage the Timer:

  • Set Timer & Close Settings - Toggles the set-up area
  • Start - Starts counting down from the current setting.
  • Stop - Pauses the timer. Click Start to continue from where it left off.
  • Reset - Returns the clock to the original setting. If you start the countdown timer before the class is ready, click Stop, and then Reset to start over.
  • Clear - Clears all inputs and sets the timer to zero.
